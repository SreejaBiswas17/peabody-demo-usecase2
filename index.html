SAP Entry Workflow Checklist

F-30 Steps
1
Log into SAP S/4 PR3
Access SAP S/4 HANA Production system PR3 and navigate to T-code F-30
2
Enter Document Header
Enter Document Date, Translation Date, Posting Date, Currency (USD), Company Code, Type AB, Reference # (initials), Doc Header Text: "CSP I/C Invoice Clearing"
3
Check Transfer Posting with Clearing
Ensure the "Transfer Posting with Clearing" option is checked before proceeding
4
Enter First Line Item (AP)
Input Posting Key and Account, press Enter. Enter Amount, Profit Center, Text. Click "More" to add Partner Profit Center
5
Process Open Items (Vendor — K)
Click "Process Open Items", enter Company Code and Account from AP Tab. Set Account Type = K. Press Enter. Unselect all items, match amounts with AP tab, press Save
6
AR Tab Entry Processing
Repeat same steps for AR tab. Exception: Enter Tax Code from Tax Code column. Set Account Type = D (Customer) before clearing Customer open items
7
Final Save & Document Number
Press Save to post the document. Record the SAP document number generated.
